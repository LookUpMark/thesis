# Notebooks

This folder contains Jupyter notebooks and related artifacts used for demos, experiments, and result exploration.

## Structure

- `00_interactive_demo.ipynb` — main interactive demo notebook.
- `ablation/` — ablation-related artifacts:
  - `ABLATION_TASK_LIST.md` — checklist for ablation work.
  - `ablation_results/` — output artifacts from ablation runs.
- `e2e/` — end-to-end notebooks and generated artifacts.
  - `generated/` — generated artifacts from the e2e notebooks.
- `unit/` — unit-test style notebooks organized by module.
- `results/` — generated reports and previews:
  - `RESULTS.md`, `RESULTS_02.md`
  - `kg_preview.html`, `kg_preview.png`
- `assets/` — static assets used by notebooks (e.g., JS libs).
